##### What is a statement?

A statement is a type of instruction that causes a computer program to perform some action



##### What is a function?

A function is a collection of statement that get executed sequentially in order (from top to bottom)



##### What is the name of the function that all programs must have?

All programs must have the function **main** 



##### What happens when a program is run?

The statements inside of main are executed in a sequential order



##### What symbol are statements in C++ often ended with?

A semi colon



##### What is a syntax error?

When your programme deviates from the syntax of the language, the compiler will issue a syntax error
